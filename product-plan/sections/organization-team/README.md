# Organization & Team

## Overview

A simple page introducing ГО "Розум" — its mission and purpose — followed by a compact list of expert team members with their names, roles, and expertise domains.

## User Flows

- View organization mission and description
- View team member list with names and roles

## Design Decisions

- Clean header with organization name and mission tagline
- Mission displayed in cyan accent color
- Description in readable paragraph format
- Team list in a card container
- Each team member shows name, role, and domain badge
- Compact, scannable layout

## Data Used

**Entities:**
- Organization (name, mission, description)
- Expert (id, name, role, domain)
